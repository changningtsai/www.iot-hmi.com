Ubique Panel

Traditionally, HMI (Human-Machine Interface) connected to the main control unit via UART port is the most universal and popular type of HMI. This HMI is to present the information clearly for humans’ consumption and take human’s responses back to the main control unit. In other words, this HMI serves as an information gateway without assuming any control functions. Therefore the graphic design is critical for packaging the information to facilitate effective flow of information. In order to work with various types of the main control unit, the HMI must support customer-defined communication protocols. The implementation of this self-defined protocol must be easy and can be done in a GUI development environment. Ubique Panel is designed to meet these constraints and at the same time enables system developers quickly design and develop the HMI.

  • An easy-to-use integrated development environment (IDE)
  • Extensive widget and wizard libraries
  • What-you-see-is-what-you-get (WYSWYG) system-level simulation environment
  • Self-defined communication protocol running on UART
  • Support expansion capability with common communication interfaces
  • Support multiple physical presentations of user interface
