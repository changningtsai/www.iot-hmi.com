ADE

ADE is the integrated development platform that bring collaboration, development and communication together in a single easy-to-use graphic environment. It supports WYSWYG (What You See is What You Get) simulation that can show the result of your project on the PC with the same aspect ratio and resolution of the target HMI. ADE allows user to implement control function by using built-in graphic interface. Sophisticated users can even leverage the programing environment of ADE to realize advanced control functionalities. In addition, ADE provides professional grade widget and wizard libraries that can significantly shorten the development cycle. These libraries offer excellent visual effect. ADE also offers the simulation capability that allows users to define the stimulus from various communication or expansion interfaces.

In order to facilitate the communication between different group of stake owners during the entire project development cycle, ADE generates documents with hyperlinks during the early design phase for the communication between developers and end-users. ADE generates detailed document under PDF or MS Word format that captures the essence of the entire HMI design and these documents can future being used as user manuals or data sheets. The capability for ADE to generate documents in every development stage is to enhance the collaboration among stakeholders and to shorten the time for documentation.

  • What is See is What You Get (WYSWYG)
  • Provide professional wizard and widget libraries with excellent visual effects
  • Support user-defined wizard and widgets across projects
  • Provide professional templates to shorten development time
  • Support graphical control development environment
  • Provide programming environment for implementing control functions
  • Simulation capability with user-defined stimulus
  • Documentation generation for the project stakeholders
